# XCloud App

XCloud App : 适合初学者，喜欢的小伙伴可以点上边的Star支持一下哦

***

### 项目架构

#### App

* Android Java语言
* 三方库
    * okhttp: 3.14.2
    * okio: 1.17.4
    * Jackson

#### 后台

* SpringBoot
* MyBatis
* MySQL
* Redis:缓存
* Druid:阿里高性能数据库链接池
* 阿里OSS:文件服务器 
* ThymeLeaf:模板引擎
* Nginx:反向代理
* Tomcat
* Maven
* jackson
* 二维码工具类
